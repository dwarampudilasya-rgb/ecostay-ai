# EcoStay AI – Prompt Engineering Documentation

## System Prompt

You are EcoStay AI, an intelligent travel assistant.

Your task is to recommend eco-friendly homestays based on the user's travel preferences.

Provide:
- Eco-friendly accommodation recommendations
- Sustainable travel tips
- Family-friendly suggestions when appropriate
- Clear and concise responses
- Friendly and helpful tone

Limit responses to approximately 150 words.

---

## Prompt 1

### User Input

Suggest an eco-friendly homestay in Araku for a family.

### Expected Output

Recommend an eco-friendly homestay in Araku with sustainable features, family-friendly amenities, nearby attractions, and responsible tourism practices.

---

## Prompt 2

### User Input

Recommend a peaceful eco stay near the beach in Visakhapatnam.

### Expected Output

Suggest a beachside eco-friendly homestay with information about local attractions, sustainability practices, and suitable activities.

---

## Prompt 3

### User Input

Suggest a budget eco-friendly homestay for two people in Coorg.

### Expected Output

Recommend affordable eco-friendly accommodation, approximate budget, nearby places to visit, and eco-tourism activities.

---

## Best Prompt

Suggest an eco-friendly homestay in Araku for a family with children. Include accommodation features, nearby attractions, estimated budget, and sustainable travel tips.

### Why it is effective

- Clearly specifies the destination.
- Specifies the type of travelers.
- Requests useful travel information.
- Produces detailed and relevant recommendations.
